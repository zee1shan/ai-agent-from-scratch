# AI Agent From Scratch

Build a production-inspired AI Agent framework from scratch using Python.

The goal of this project is to understand how modern AI agents work internally instead of relying on frameworks such as LangChain or LangGraph.

## Features (Planned)

- LLM abstraction
- Agent architecture
- Tool system
- Automatic tool discovery
- Pydantic argument validation
- Tool calling
- Conversation memory
- Planning
- RAG integration
- FastAPI API
- Docker support
- MCP integration
- LiveKit Voice Agent

## Tech Stack

- Python
- Pydantic
- Ollama
- FastAPI
- ChromaDB
- Docker

## Learning Goals

- Build an AI Agent from first principles.
- Understand tool calling and function execution.
- Design extensible software architecture.
- Learn production-ready backend development.
- Understand the concepts behind LangChain and LangGraph before using them.
